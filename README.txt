Desafio de uso da biblioteca Pandas Python.

Aulas práticas que geraram 3 arquivos com código Python usando a biblioteca Pandas.

Foi gerado uma imagem png de um gráfico e um arquivo csv dos dados trabalhados.

Também há os arquivos excel e o arquivo csv que foram utilizados.

O primeiro arquivo gerado utilizando o Pandas foi Introducao_biblioteca_Pandas.ipynb e utilizou o arquivo Gapminder.csv como base de dados.

O segundo arquivo gerado utilizando o Pandas foi trabalhando_planilhas_excel.ipynb e utilizou as planilhas Aracaju.xlsx, 
Fortaleza.xlsx, Natal.xlsx, Recife.xlsx e Salvador.xlsx como base de dados.

O terceiro arquivo gerado utilizando Pandas foi Analise_Exploratoria.ipynb, utilizou a planilha AdventureWorks.xlsx como base de dados
e que gerou um novo arquivo de dados chamado df_Venda_Novo.csv.
